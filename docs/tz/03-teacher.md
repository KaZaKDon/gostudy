# 03-teacher.md — Кабинет преподавателя

## Статус документа

Этап 1: ТЗ для верстки фронта кабинета преподавателя. Используются временные данные, API и БД не проектируются, экран «Классы» — заглушка, виртуальный класс не реализуется.

---

# Анкета преподавателя после регистрации

## Личные данные

- Фамилия;
- Имя;
- Отчество;
- Год рождения / дата рождения;
- Пол.

## Местоположение

- Страна;
- Регион;
- Город / населенный пункт;
- Часовой пояс.

## Контакты

- Телефон;
- Email;
- Основной мессенджер.

## Профессиональная информация

Статус: Учитель школы, Преподаватель колледжа, Преподаватель ВУЗа, Частный репетитор, Другое.

Образование: учебное заведение, специальность, год окончания.

Опыт работы: текстовое описание, периоды работы можно предусмотреть позже.

Предметы: множественный выбор.

Направления: ОГЭ, ЕГЭ, Школьная программа, Иностранные языки, Начальная школа, Подготовка к поступлению, Другое.

О себе: свободное текстовое поле.

Медиа: фото профиля.

Документы: дипломы, сертификаты, курсы. Документы не показываются публично по умолчанию. Родителям могут предоставляться по запросу и только с согласия преподавателя.

---

# Стоимость занятий

Стоимость не хранится в анкете преподавателя.

Стоимость относится к услугам / урокам: предмет, формат, длительность урока, количество занятий в неделю, стоимость урока, стоимость курса.

Цена конкретного урока фиксируется на момент записи.

# Экран «Расписание»

Первый экран после входа преподавателя.

## Основной вид

```text
Понедельник     15 сентября     3 урока     с 10:00
Вторник         16 сентября     2 урока     с 12:00
Среда           17 сентября     5 уроков    с 09:00
Четверг         18 сентября     4 урока     с 11:00
Пятница         19 сентября     3 урока     с 10:00
Суббота         20 сентября     1 урок      с 13:00
Воскресенье     21 сентября     выходной
```

Клик по дню открывает список уроков выбранного дня.

Клик по уроку открывает карточку урока: ученик, предмет, тема, время, статус оплаты.

Кнопки карточки урока: Открыть класс, Открыть ученика, Перенести, Отменить.

Верхняя панель: Добавить урок, Предыдущая неделя, Следующая неделя.

---

# Экран «Классы»

Заглушка: «Раздел находится в разработке».

---

# Экран «Домашние работы»

Главный объект раздела — конкретное домашнее задание, а не ученик.

Второй сайдбар:

```text
На проверке (3)
Просроченные (2)
В работе (8)
Проверенные
```

Рабочая область — список работ выбранного статуса.

Клик по работе открывает карточку задания: ученик, предмет, тема, дата выдачи, срок сдачи, ответ ученика, прикрепленные файлы / изображения, комментарий.

Действия: Проверено, Отправить на доработку, Комментарий, Открыть ученика, История по теме.

---

# Экран «Журнал»

Привычный для преподавателя журнал.

Второй сайдбар — список учеников.

Рабочая область — таблица по выбранному ученику:

```text
Дата         Тема                 ДЗ         Оценка
15.09        Уравнения            Да         5
22.09        Квадратные           Да         4
29.09        Функции              Да         5
```

Клик по строке открывает подробности урока: дата, тема, раздел учебника, план урока, домашнее задание, оценка если используется, комментарий для ученика, внутренняя заметка преподавателя.

Внутренняя заметка преподавателя не попадает в дневник ученика.

---

# Экран «Ученики»

Основной сайдбар остается. Рабочая область делится на второй сайдбар со списком учеников и правую область с информацией выбранного ученика.

Второй сайдбар: поиск и список учеников.

Карточка ученика: ФИО, класс, предмет, следующий урок, статус оплаты.

Табы карточки: Общее, История занятий, Домашние задания, Рабочая программа, Материалы, Оплаты, Родители, Отзывы.

Общее: телефон ученика, телефон родителя, мессенджер, цель обучения.

История занятий: дата, тема, учебник, раздел, план урока.

Рабочая программа: индивидуальный маршрут обучения, цель, срок, этапы, прогресс.

Материалы: ссылки на материалы из общей библиотеки, не копии.

---

# Экран «Материалы»

Верхние табы: Учебники, Тренажёры, Дополнительные материалы.

Учебники: второй сайдбар по предметам; рабочая область со списком учебников; действия просмотр и назначить ученикам.

Тренажёры: второй сайдбар по предметам; название, описание, ссылка, назначить ученикам.

Дополнительные материалы: авторские материалы преподавателя — методички, пособия, сборники задач, рабочие тетради, брошюры, шпаргалки.

Доступ: бесплатно или платно. Действия: загрузить, редактировать, назначить ученикам, указать стоимость, скрыть / показать.

В карточках учеников, уроках и домашних заданиях хранятся ссылки на материалы, а не копии.

---

# Экран «Оплаты»

Показывает финансы преподавателя по ученикам: кто оплатил, кто не оплатил, сколько получено, сколько ожидает подтверждения, сколько ожидает выплаты.

Второй сайдбар — список учеников.

Карточка ученика: ФИО, предмет, дата начала обучения.

Тип оплаты первой версии: поурочно, курсом. Абонемент закладывается в архитектуру, но скрыт.

Поурочно: стоимость урока, количество проведенных уроков, оплачено, ожидает подтверждения.

Курсом: стоимость курса, оплачено, остаток.

История оплат: дата, урок, сумма, статус.

Сводка: всего оплачено, проведено уроков, ожидает выплаты.

---

# Экран «Сообщения»

Служебная переписка внутри платформы.

Второй сайдбар: список диалогов, поиск, фильтры Все / Ученики / Родители.

Рабочая область: переписка, сообщения собеседника слева, преподавателя справа.

Панель ввода: текст, смайлы, файл, изображение, отправить.

Первая версия: текст, смайлы, изображения, файлы. Не входят: голосовые, видеосообщения, звонки.

---

# Экран «Отзывы»

Отзывы учеников и родителей о работе преподавателя.

Второй сайдбар — список авторов отзывов.

Карточка отзыва: автор, дата, оценка, текст, ответ преподавателя.

Отрицательные отзывы допускаются. Преподаватель не может удалять отзывы, но может ответить или оспорить. Спорные отзывы рассматриваются администрацией.

Отзыв может оставить только ученик или родитель, имеющий реальную связь с преподавателем через платформу. Можно предусмотреть минимум 5 проведенных занятий.

---

# Экран «Настройки»

Второй сайдбар: Профиль, Контакты, Безопасность, Уведомления, Документы, Платежные данные, Публичная анкета.

Профиль: ФИО, дата рождения, пол, город, часовой пояс, фото.

Контакты: телефон, email, мессенджер.

Безопасность: сменить пароль, выйти со всех устройств, удалить аккаунт.

Уведомления: уроки, домашние работы, сообщения, оплаты, отзывы. Каналы: внутри сайта, email, Telegram позже.

Документы: диплом, сертификаты, курсы, подтверждение образования. Доступ родителям только по запросу и с согласия преподавателя.

Платежные данные: карта / счет для выплат, статус проверки, история выплат.

Публичная анкета: о себе, образование, опыт, предметы, направления, фото, показывать профиль да / нет. Стоимость здесь не редактируется.
