# GoStudy — Описание проекта

## Идея

GoStudy — платформа для индивидуального онлайн-обучения между преподавателями и учениками.

Основной акцент:

* индивидуальные занятия
* безопасность общения
* участие родителей
* встроенный учебный кабинет

---

# Роли

## Ученик

Может:

* искать преподавателей
* общаться
* подавать заявки
* проходить обучение
* оставлять отзывы

---

## Родитель

Может:

* читать переписку ребенка
* писать преподавателю от своего имени
* контролировать обучение

---

## Преподаватель

Может:

* принимать заявки
* вести учеников
* назначать уроки
* выдавать домашние задания
* получать отзывы

---

# Основная логика

Поиск преподавателя
