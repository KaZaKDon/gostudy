# Назначение

GoStudy Admin — административная панель платформы GoStudy.

Админка полностью независима от пользовательской части сайта.

Вход осуществляется через

```text
/admin/login
```

---

# Роли

## Administrator

Полный доступ.

Может

* изменять роли;
* блокировать;
* архивировать;
* восстанавливать;
* подтверждать преподавателей;
* работать с документами;
* работать с платежами;
* изменять настройки.

---

## Moderator

Ограниченный доступ.

Может

* просматривать пользователей;
* проверять документы;
* проверять преподавателей;
* работать с отзывами;
* работать с жалобами.

Не может

* менять роли;
* блокировать пользователей;
* архивировать пользователей;
* изменять настройки.

# Accounts

Назначение раздела

Технический реестр пользователей.

Раздел не содержит анкет.

Показывает

* ФИО;
* Email;
* Телефон;
* Роль;
* Статус;
* Последний вход.

Действия администратора

* Просмотр
* Изменение роли
* Блокировка
* Разблокировка
* Архивирование
* Восстановление
* Переход в профиль

Действия модератора

* Только просмотр.

---

# Архив пользователей

Физическое удаление пользователей запрещено.

Используется жизненный цикл

```text
active

↓

blocked

↓

deleted (архив)
```

Архив не удаляет

* уроки;
* домашние задания;
* сообщения;
* платежи;
* отзывы.

Аккаунт может быть восстановлен.

# UI Library

Все универсальные компоненты располагаются

```text
src/admin/components/ui/
```

Business Components располагаются отдельно.
