# Что сознательно НЕ делали

Не стали делать:

- модерацию дипломов;
- модерацию сертификатов;
- модерацию фото;
- модерацию видеовизитки.

Причина:

Во фронте GoStudy ещё отсутствует загрузка файлов.

Сначала необходимо реализовать регистрацию, заполнение профиля и загрузку документов.

После этого вернуться в админку и сделать модерацию документов.

без проверки роли.
